# Cupidon Bot 💌

Bot Telegram qui pose une question romantique chaque jour à votre couple, avec des réponses adorables générées par IA.

## Commandes

- `/question` — Pose une question du jour
- `/souvenirs` — Liste tous les souvenirs collectés
- `/souvenir` — Affiche un souvenir au hasard
- `/stats` — Statistiques du couple

## Déploiement

1. Créez le bot sur [@BotFather](https://t.me/BotFather)
2. Obtenez une clé API [Groq](https://console.groq.com) (gratuite)
3. Pushez sur GitHub et déployez sur [Railway](https://railway.app)
4. Ajoutez les variables d'environnement : `BOT_TOKEN`, `GROQ_API_KEY`, `CHAT_ID`
